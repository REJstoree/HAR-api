# WhatsApp API Multi (Baileys)

1. npm install
2. npm start
3. Abra http://localhost:3000
4. Crie uma sessão (nome) e escaneie o QR que aparece no painel
5. Envie mensagens via painel ou via API POST /api/message

Deploy no Cyclic:
- Crie um repositório no GitHub com esses arquivos.
- Acesse https://app.cyclic.sh → Deploy → escolha o repositório.
- Cyclic detecta Node e executa `npm install` + `npm start`.
